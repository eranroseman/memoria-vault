**Goal:** Land the alpha.21 I1 skeleton — complete the `empirical_event.v1` client shape, emit honest server-side `disposition.v1` and `read-observed.v1` journal events from one real decision path and one real read path, and plant the shadow-first `production_enabled` flag — so the non-backfillable telemetry seam exists before beta.1.

**Architecture:** Four self-contained changes on existing seams. The empirical-event validator (`engine/empirical_events.py`) gains two client-shape fields plus two small server-event validators. A reusable `emit_disposition_event` seam (`runtime/operations.py`) is wired into `resolve_attention` (`runtime/integrity.py`). `answer_query` (`runtime/search_index.py`) appends a `read-observed.v1` event. A `feedback.yaml` config + reader is seeded and surfaced read-only in `doctor`. All events append through the existing `trusted_writer.append_journal_event` path, which stamps the F1-true actor and `request_id` from the operation context.

**Tech Stack:** Python 3.12 stdlib + SQLite + PyYAML (already a dependency). Tests: pytest via `python3 scripts/verify`.

Spec: `docs/superpowers/specs/2026-07-14-i1-skeleton-design.md`. Milestone `0.1.0-alpha.21`; issue #1365.

## Global Constraints

It runs pytest `-m "static or unit or contract"` — so **every new test file must be registered in `tests/conftest.py` `TEST_LEVELS` at `contract`** (not `runtime`), or the gate will silently skip it. This matches the existing `test_empirical_events.py` precedent (a journal-recording test at `contract`).

- No new dependencies (Python 3 stdlib + SQLite + the already-present PyYAML).

- Actor vocabulary is fixed: `'pi' | 'agent' | 'operation' | 'integrity'`. Server-emitted events carry the actor via journal provenance (stamped by `append_journal_event`), never a synthesized field.
- PR boundary: one squash PR after Task 5, titled `feat(feedback): I1 instrumentation skeleton (#1365)`.

def validate_disposition_event(payload: dict[str, Any]) -> dict[str, Any]:
    """Return a normalized server-side disposition event or raise ``ValueError``."""
    if not isinstance(payload, dict):
        raise ValueError("disposition event payload must be an object")
    unknown = sorted(set(payload) - DISPOSITION_REQUIRED_FIELDS)
    if unknown:
        raise ValueError(f"disposition event contains unsupported fields: {', '.join(unknown)}")
    missing = sorted(field for field in DISPOSITION_REQUIRED_FIELDS if _missing(payload.get(field)))
    if missing:
        raise ValueError(f"disposition event missing required fields: {', '.join(missing)}")
    decision = _string_field("decision", payload["decision"])
    if decision not in DECISIONS:
        raise ValueError(f"decision must be one of: {', '.join(sorted(DECISIONS))}")
    return {
        "decision": decision,
        "item_type": _string_field("item_type", payload["item_type"]),
        "item_id": _string_field("item_id", payload["item_id"]),
    }
```

def validate_read_event(payload: dict[str, Any]) -> dict[str, Any]:
    """Return a normalized server-side read-observation event or raise ``ValueError``."""
    if not isinstance(payload, dict):
        raise ValueError("read event payload must be an object")
    unknown = sorted(set(payload) - READ_REQUIRED_FIELDS)
    if unknown:
        raise ValueError(f"read event contains unsupported fields: {', '.join(unknown)}")
    missing = sorted(field for field in READ_REQUIRED_FIELDS if _missing(payload.get(field)))
    if missing:
        raise ValueError(f"read event missing required fields: {', '.join(missing)}")
    workflow = _string_field("workflow", payload["workflow"])
    if workflow not in WORKFLOWS:
        raise ValueError(f"workflow must be one of: {', '.join(sorted(WORKFLOWS))}")
    return {"workflow": workflow, "staleness_hit": _bool_field("staleness_hit", payload["staleness_hit"])}
```

```python
"""Shadow-first discipline flag for I1 feedback instrumentation."""

from __future__ import annotations

from pathlib import Path

import yaml

FEEDBACK_CONFIG = ".memoria/config/feedback.yaml"


def feedback_production_enabled(vault: Path) -> bool:
    """Return True only when feedback.yaml explicitly sets production_enabled: true.

    Gates *acting* on the signal, never recording. Fails safe to False on an
    absent, unreadable, malformed, or key-missing config.
    """
    path = Path(vault) / FEEDBACK_CONFIG
    if not path.is_file():
        return False
    try:
        data = yaml.safe_load(path.read_text(encoding="utf-8"))
    except yaml.YAMLError:
        return False
    if not isinstance(data, dict):
        return False
    return data.get("production_enabled") is True
```
