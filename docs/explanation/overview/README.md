# Operational overview

Memoria is a vault-first research system. The vault holds durable knowledge, the
board holds transient work, and Hermes runs the specialist profiles that propose
changes back to the human. Explanation pages focus on how those pieces fit
together during normal use.

## The working shape

| Layer | Role |
| --- | --- |
| Vault | Durable notes, catalog entities, projects, spaces, and system artifacts. |
| Board | Short-lived worker-card state for delegated tasks. |
| Inbox | PI-facing action cards that need a decision now. |
| Profiles | The Co-PI and four background lanes that read, propose, map, draft, and verify. |
| Operations | Deterministic tools that ingest, lint, cluster, sweep, and measure. |
