# cmdr (Commander)

Recommended. The command palette already exposes every Memoria action; Commander's value is putting the five-to-ten most frequent ones on physical buttons. Load-bearing setting: none — but the recommended set is `Memoria: capture fleeting`, `Memoria: ask about this note`, `Memoria: new project`, `Memoria: lint this note`, `Memoria: approve all link suggestions`. If the button strip grows past ten, it stops being faster than the palette and the discipline has failed.

The full Memoria command catalog (standard commands across Capture / Processing / Interactive retrieval / Project / Maintenance / Lens categories) lives in [command-catalog.md](../../reference/command-catalog.md).
