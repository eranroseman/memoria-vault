# Retired ADRs

`docs/adr/` is no longer the live decision-record system.

Historical ADRs remain evidence in git history and in the frozen release
chapters under
[design history](https://github.com/eranroseman/memoria-vault/blob/main/design-history/README.md).
Current decision capture happens in the active release workspace as dated
Y-statements, then closes into `design-history/` with the release.

Use:

- [Design history](https://github.com/eranroseman/memoria-vault/blob/main/design-history/README.md)
  for frozen release chapters and maintenance rules.
- [Design arcs](https://github.com/eranroseman/memoria-vault/blob/main/design-history/arcs.md)
  for the current released line and pending unreleased questions.
- `releases/<version>/decisions.md` on the `scratch` branch for decisions still
  being made.
