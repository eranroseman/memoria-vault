# Fix a stale .bib

This alpha.10 troubleshooting guide is retired in alpha.11. The old
`.memoria/memoria.bib` auto-export path no longer exists.

`references.bib` is generated from checked source Concepts. If it is stale,
enqueue `regenerate-references-bib` or `regenerate-tracked-projections` through
the worker.
