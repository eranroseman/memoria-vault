# Retired sample vault

The alpha.10 sample vault is no longer part of the shipped alpha.11 system.
There is nothing to load or remove: the starter vault installs without bundled
sample notes, and QuickAdd does not register sample-vault commands.

Use a fresh vault for alpha.11. Capture your own notes from Obsidian or enqueue
source work from the Memoria Inspector control panel; the worker/check loop owns
machine writes and promotion.
