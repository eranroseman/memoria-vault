# Compose

Day-to-day tasks for turning accumulated knowledge into written output. Performed inside Obsidian via the Co-PI pane and the Inbox; export runs in the terminal.

> The full project workflow ships until the alpha.3 Project release — see [Start a writing project](start-a-writing-project.md) for what to do meanwhile.

| Guide | What it covers | Cycle phase |
| --- | --- | --- |
| [Start a writing project](start-a-writing-project.md) | v0.1.0-alpha.3 scope, and the v0.1.0-alpha.2 path meanwhile | — |
| [Assess your corpus](assess-your-corpus.md) | Delegate a `map` task: dense clusters, thin coverage, gap cards | Assess |
| [Frame a project](frame-a-project.md) | Competing outlines from the `draft` lane; choose one framing | Frame |
| [Use canvas for argument mapping](use-canvas-for-argument-mapping.md) | Arrange claims spatially to find structure before prose | Sketch |
| [Query the vault](query-the-vault.md) | Ask the Co-PI a research question, grounded in your notes | Query (aid) |
| [Draft with the Writer](draft-with-writer.md) | Delegate sections to the `draft` lane; edit the result | Draft |
| [Verify and revise a draft](verify-and-revise.md) | Delegate `verify` passes; act on flag cards; loop | Verify |
| [Work the review queue](work-the-review-queue.md) | Sweep the Inbox: proposals, flags, and held gated writes | Review (gate) |
| [Export a draft](export-a-draft.md) | Pandoc export to Word, PDF, or plain Markdown | Export |
| [Create a code artifact](create-a-code-artifact.md) | Delegate the `code` lane; hand off to an external coding agent | Code (branch) |
