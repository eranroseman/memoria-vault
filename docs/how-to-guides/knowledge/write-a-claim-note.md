# Write a claim note

This alpha.10 guide is retired in alpha.11. The old `notes/claims/` claim-note
surface and the `Memoria: write claim note` / `Memoria: create linked claim
note` commands no longer ship.

Use `Memoria: capture note` for a new PI-owned note, or edit checked Concepts
directly when the claim belongs in an existing note.
