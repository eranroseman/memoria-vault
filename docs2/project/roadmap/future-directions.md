# Future directions

Deferred capabilities: designed and endorsed but not yet implemented.

- **Discovery loop** — nightly Librarian run, bounded batch, writes to inbox only.
- **CiteME-style Verifier harness** — numeric benchmark for citation verification quality.
- **Chain-of-Evidence claim taxonomy** — `citation` / `numerical` / `methodological` / `conclusion` types with per-type integrity checks.
- **Scenario-typed retrieval** — `similar`, `cross-domain`, `counter-intuitive` wikilink relations beyond `supports` / `contradicts`.
- **Agent-consensus pre-filter** — multiple agents corroborate before reaching human review queue.
- **Exploration-trace capture** — preserve rejected directions and dead ends alongside approved knowledge.
- **Coder lane experiment loop** — autonomous keep/revert inside the Coder lane against a pre-defined test suite.

See [../../explanation/architecture/why-not-autonomous.md](../../explanation/architecture/why-not-autonomous.md) for the autonomy boundary these directions operate within.
