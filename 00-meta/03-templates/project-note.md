# `project-note` template

Coordinates a research project, its scope, milestones, and active decisions. Lives in `40-workbench/`. Human-authored.

## Frontmatter

```yaml
---
title: ""
type: project-note
lifecycle: current
project_phase: active      # planning | active | paused | complete
phase: planning
start_date:
due_date:
scope: ""
research_question: ""
moc: []
related_notes: []
related_sources: []
outputs: []
schema_version: 1
created:
updated:
---
```

## Body

```md
# Goal
What this project is trying to accomplish.

# Scope
- In scope.
- Out of scope.
- Constraints.

# Progress
- Current status.
- Next milestone.
- Open decisions.

# Links
- Relevant paper notes.
- Relevant claim notes.
- Relevant code notes.
```
